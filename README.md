# 🎵 Аудіо Спектроаналізатор

**Аудіо спектроаналізатор** — це пристрій, який візуалізує навколишній звук у реальному часі. 

За допомогою мікрофона він вловлює звуки із зовнішнього середовища (музику з колонок, голос, шум). Мікроконтролер оцифровує цей аналоговий сигнал, обробляє його за допомогою алгоритму Швидкого перетворення Фур'є (FFT) і виводить результат — спектр частот від басів до високих нот — на чотири світлодіодні матриці 8х8 у вигляді динамічних стовпчиків.

---
## 📂 Структура проєкту
- 📂 **Analis_Audio_Signal_ARDUINO** — Початкова версія коду для плат Arduino.
- 📂 **Audio_Spectrum_Analyzer_AltiumDesigner** — Схема та трасування друкованої плати (PCB), розроблена в Altium Designer.
- 📂 **vs-code** — Оптимізований модульний C-код для мікроконтролера STM8S103F3P6, налаштований для збірки через VS Code.
